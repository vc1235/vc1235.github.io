justinmind的破解

免费的justinmind只有三十天的试用期，不过没关系，有办法在要到期的时候重置这个时间。在6.6.1版本前都是好用的.

mac os

删除个人文件夹下面的.configprops和.datastorage这两个文件

windows xp

删除C:\Documents and Settings\当前用户下的.configprops和.datastorage

windows 7/8

删除C:\Users\当前用户下的.configprops和.datastorage
